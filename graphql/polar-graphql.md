# Polar GraphQL API

Polar is a creator monetization platform for newsletters and digital products. The API covers subscriptions, products, benefits, orders, customers, and analytics for newsletter writers and indie developers.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Polar REST API
**Documentation:** https://docs.polar.sh/
